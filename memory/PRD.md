# نظام إدارة المهام والموظفين - PRD

## المشكلة الأصلية
تحويل موقع إلى تطبيقات Android/iOS/Windows مع تعديلات على نظام المهام

## التعديلات المنفذة (آذار 2026)

### 1. إصلاح أزرار الحذف ✅
- حذف المهمة - زر أحمر واضح "حذف المهمة"
- حذف الموظف - زر أحمر "حذف الموظف"
- حذف السلفة - زر أحمر "حذف"
- معالجة أخطاء Token expired

### 2. نظام الرواتب الكامل ✅
- تحديد الراتب الأساسي لكل موظف
- إضافة/حذف السلف (تُخصم تلقائياً)
- تسليم الراتب مع حساب الصافي
- سجل الرواتب المُسلّمة (جدول احترافي)
- إشعارات تليجرام

### 3. شاشة أوقات الموظفين المحسنة ✅
- تصميم احترافي مع avatar
- جدول منظم (التاريخ، الدخول، الخروج، المدة، الحالة)
- Modal احترافي لتحديد وقت العمل

### 4. ميزات إضافية ✅
- زر عرض التقرير لكل مهمة
- زر تعديل المهمة
- زيادة صلاحية Token إلى 7 أيام
- إرسال رسائل للموظفين
