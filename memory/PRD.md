# نظام إدارة المهام - PRD

## المشكلة الأصلية
المستخدم يريد تعديل صفحة المهام (admin) لإضافة:
1. عرض التقارير التي يرسلها الموظفين
2. زر تقرير بجانب كل مهمة
3. زر تعديل للمهمة
4. إشعارات تليجرام عند إكمال/عدم إكمال المهام
5. 3 تنبيهات تأخير إذا تأخرت المهمة ساعة كاملة

## ما تم تنفيذه (Jan 2026)

### 1. زر عرض التقرير
- زر "عرض التقرير" أو "لا يوجد تقرير" بجانب كل مهمة
- نافذة تفصيلية تعرض:
  - معلومات المهمة (الزبون، الهاتف، العنوان، العطل، الموظف)
  - التقرير إذا كان موجود
  - حالة المهمة (مكتملة/غير مكتملة)
  - مدة التنفيذ
  - تاريخ الإنهاء

### 2. زر تعديل المهمة
- زر "تعديل" أزرق بجانب كل مهمة
- نافذة تعديل تحتوي على:
  - اسم المشترك
  - رقم الهاتف
  - العنوان
  - وصف العطل
  - اختيار الموظف المكلف

### 3. حفظ التقارير
- تعديل API `/api/tasks/{task_id}/complete` لحفظ:
  - نص التقرير (report_text)
  - حالة النجاح (success)
  - صور التقرير (images)
  - مدة التنفيذ (duration_minutes)

### 4. إشعارات التليجرام
- إشعار للمدير عند إكمال/عدم إكمال المهمة
- يشمل: اسم الموظف، الزبون، العنوان، التقرير، المدة

### 5. تنبيهات التأخير
- نظام خلفي يفحص المهام كل 10 دقائق
- إذا تأخرت المهمة ساعة: يرسل حتى 3 تنبيهات
- التنبيهات تُرسل للموظف والمدير

## الملفات المعدلة
- `/app/frontend/src/pages/AdminDashboard.js`
- `/app/backend/server.py`

## APIs الجديدة
- `PUT /api/tasks/{task_id}` - تحديث المهمة
- `POST /api/tasks/{task_id}/complete` - إكمال المهمة مع التقرير

## التقنيات المستخدمة
- Frontend: React.js
- Backend: FastAPI (Python)
- Database: MongoDB
- Notifications: Telegram Bot API
